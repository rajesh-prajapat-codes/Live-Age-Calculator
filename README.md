# Live Age Calculator

A modern and responsive Live Age Calculator built using HTML, CSS, and JavaScript.

Users can enter their date and time of birth and instantly see their live age updating every second in:

- Years
- Months
- Days
- Hours
- Minutes
- Seconds

## How It Works

1. User enters their Date of Birth
2. JavaScript calculates the difference between:
   - Current Time
   - Birth Time
3. `setInterval()` updates the age every second
